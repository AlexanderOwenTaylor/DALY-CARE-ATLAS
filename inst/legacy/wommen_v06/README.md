# Legacy WoMMen V06 Cartography Reference

This directory preserves selected WoMMen V06 cartography and atlas files as
provenance/reference evidence only.

Do not source these files from the `dalycareatlas` runtime. The active package
implementation ports the useful ideas into clean modules under `R/`, with
aggregate-only output guarantees and test coverage.

Included reference material:

- `RUN_CARTOGRAPHY_ATLAS.R`
- `WoMMen_code/code/000_dalycare_cartography_consolidated.R`
- `WoMMen_code/code/001_dalycare_atlas_updater.R`
- `site/DALYCARE_atlas_AOT_V35.html`
- `site/DALYCARE_atlas_payload.js`
- selected V06 patch logs and README notes
