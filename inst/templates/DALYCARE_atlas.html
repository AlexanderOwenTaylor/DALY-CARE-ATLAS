One click

RStudio Console Run

Open the repository directory in RStudio and source the one-click runner.

source("RUN_DALYCARE_ATLAS.R")

The runner catches errors, stores diagnostics in the session, writes to atlas_runs/, and never calls quit().

CLI

Terminal Run

The command-line interface stays unchanged for reproducible scheduled or scripted runs.

Rscript scripts/run_atlas.R /path/to/DALY-CARE-ATLAS config/source-map.dalycare64.production.tsv atlas_runs report

Access check

Preflight Without Patient Rows

Validate required files, the 64-resource production source map, and the resolver dry-run without writing patient-level data.

Rscript scripts/check_dalycare_bootstrap.R /path/to/DALY-CARE-ATLAS

The standard loader is auto-detected at /ngc/projects2/dalyca_r/clean_r/load_dalycare_package.R when present.
