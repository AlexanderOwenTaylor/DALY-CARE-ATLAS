Quick Start

Run the DALY-CARE-aware atlas with the package bootstrap and curated source map.

1. Configure DALY-CARE package access

export DALYCARE_PACKAGE_ROOT=/path/to/dalycare_package
export DALYCARE_BOOTSTRAP_PATH=/path/to/DALY-CARE-ATLAS/inst/templates/dalycare_bootstrap.R

Live database credentials remain controlled by the user's NGC db_access.R setup.

2. Preflight without live data

Rscript scripts/check_dalycare_bootstrap.R /path/to/DALY-CARE-ATLAS

This validates the bootstrap and source map without emitting patient-level data.

3. Run the atlas

Rscript scripts/run_atlas.R /path/to/DALY-CARE-ATLAS config/source-map.dalycare.tsv atlas_runs report

Outputs are CSV evidence files plus this HTML viewer and its external JS payload.
