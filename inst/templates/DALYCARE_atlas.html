Quick Start

Run the DALY-CARE-aware atlas with the package bootstrap and curated source map.

1. Confirm production access

# Standard loader is auto-detected when present:
# /ngc/projects2/dalyca_r/clean_r/load_dalycare_package.R
# Credentials stay in /ngc/people/<user>/db_access.R

Set DALYCARE_BOOTSTRAP_PATH only when using a non-standard loader.

2. Preflight without live data

Rscript scripts/check_dalycare_bootstrap.R /path/to/DALY-CARE-ATLAS

This validates bootstrap, db_access.R, DB catalog resolution, and the source map without emitting patient-level rows.

3. Run from RStudio Console

source("scripts/run_atlas.R")
result <- run_atlas_from_source(
  source_map_path = "config/source-map.dalycare.tsv"
)

Runs refuse misleading empty live atlases unless DALYCARE_ATLAS_ALLOW_EMPTY_LIVE_RUN=TRUE is set for a dry run.

4. Run from a terminal

Rscript scripts/run_atlas.R /path/to/DALY-CARE-ATLAS config/source-map.dalycare.tsv atlas_runs report

Outputs are CSV evidence files plus this HTML viewer and its external JS payload.
