# PDSA Cycle 2 Study Template

## Prediction

If the atlas opens with a role-specific briefing, a shared trust legend, and explicit count/status semantics, internal reviewers should reach the correct first panel faster without losing access to raw technical lineage.

## Study Questions

- Can a PI identify what can be used now and what needs validation first?
- Can a data manager identify unresolved source or field actions?
- Can disease researchers route to PCD/MM, CLL, lymphoma, and other LPD evidence without opening raw lineage first?
- Can new employees complete the first 3 safe tasks using aggregate-only paths?
- Can clinicians/managers read a one-page print briefing without encountering causal or recommendation claims?

## Measures

- Static regression tests.
- Preservation snapshots.
- Visual QA screenshots across desktop, laptop/tablet-like, phone, and print briefing views.
- Manual review of prohibited language and privacy safeguards.
