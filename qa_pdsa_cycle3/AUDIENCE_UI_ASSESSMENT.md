# PDSA Cycle 3 Audience UI Assessment

## Aim

Reduce reviewer friction caused by raw cartography labels, misleading mixed-denominator bars, duplicate aliases, and free-text/code inventory mixing.

## Assessment

The issue was presentation and interpretation friction, not data absence. The atlas already contains rich aggregate evidence, but several panels made reviewers decode source prefixes, raw variable names, repeated aliases, and unmapped code rows before seeing the clinical/source concept.

## Preservation Position

- All aggregate rows remain available.
- Primary views are human-readable summaries.
- Raw technical lineage is secondary, collapsed where appropriate, and exportable.
- No count logic, source discovery, production queries, payload arrays, privacy safeguards, or small-cell rules are changed.
